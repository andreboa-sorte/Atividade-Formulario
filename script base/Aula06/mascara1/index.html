Máscara em JavaScript

Para campos númericos, como Telefone, Celular, CPF, CNPJ, RG etc.

Exemplos

Como usar?

Para usar é muito simples, basta adicionar o script "mascara.js" no head do arquivo HTML.

No campo do formlário inclua:

onkeyup="mascara('(##)#####.####',this,event,true)"

Você pode formar as máscaras que quiser, onde aparece o caractere # será substituído pelo número digitado pelo usuário. Os outros símbolos aparecerão como você colocou nas máscaras.
